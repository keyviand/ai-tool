# AI HR Candidate Filter

This is a pure frontend version of the AI HR Candidate Filter app. It works on GitHub Pages because it only uses HTML, CSS, and JavaScript.

## Features

- Enter a job title
- Enter required skills
- Add multiple candidates
- Paste resume text
- Generate match scores
- Rank candidates
- Show matched and missing skills

## Files

- `index.html` - Main webpage
- `style.css` - Styling
- `script.js` - Candidate scoring logic

## How to Use Locally

Open `index.html` in your browser.

## How to Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html`, `style.css`, and `script.js`.
3. Go to repository Settings.
4. Click Pages.
5. Under Branch, choose `main`.
6. Click Save.
7. Open the GitHub Pages link after it publishes.

## Ethical Note

This project should support HR review and should not replace human hiring decisions.
