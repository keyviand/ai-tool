# AI HR Candidate Filter

A simple Flask web application that helps HR teams filter job candidates using AI-style skill matching.

## Features

- Enter job title
- Add required skills
- Paste candidate resume text
- Generate candidate match scores
- Rank candidates
- Display matched and missing skills

## Tech Stack

- Python
- Flask
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```

## Ethical Note

This application should support HR review and not replace human hiring decisions.
